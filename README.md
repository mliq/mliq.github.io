# [mliq.github.io](http://mliq.github.io)

Michael Liquori — Senior Staff Engineer (Front-End).

Quickstart
- Build styles: `npm run build` (or `npm run watch` during development)
- Serve locally: `npm run serve` then open `http://localhost:8000`
- Petfinder capture: see `CAPTURE.md` and scripts in `package.json`

Notes
- Edit styles in `public/styles/custom.scss`. The file `public/styles/custom.css` is compiled and will be overwritten by the build.
