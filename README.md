#Scope of Work: “Michael Liquori”
####Client: Michael Liquori
####Developer: Michael Liquori

##Overview
Michael Liquori (henceforth, “Developer”) will create a web portfolio and contact site for Michael Liquori (“Client”).

The site must demonstrate knowledge of jQuery and Bootstrap, and be hosted on heroku using node.js.

It will display contact information and project information in a minimalist style.

##This Document
This scope of work agreement supersedes all supporting documents and other verbal and written agreements as to what will and will not be done.

##Time Requirements
The project will be completed by 12 pm Friday, May 8, 2015.
	
##Components / Features

###Creative
Client will provide creative, including all copy, graphics, color scheme, and fonts. 

###User Interaction & Capabilities
All links will change color on hover. Link targets will be animated when opening or closing. There will be ‘X’ glyphs in the upper-left corner for closing.

###Layout
See wireframe at: https://wireframe.cc/jlkFKf 
There will be a nav bar showing “Contact” and “Projects” options.
Clicking on “Projects” will bring up a sub-nav menu for listing projects, clicking on them will bring up a project “showcase” box.
Clicking on “Contacts” will bring up a sub-nav menu with various links.
There will be close buttons that can close the sub-nav menu, the contact sub-nav menu, and project showcases.

###User Tracking / Analytics
Google Analytics is installed.

###Mobile / Netbook / Tablet Access
As time allows, attempts will be made to test on various screen sizes, but no guarantees are made due to the short timeframe of the project.

##Implementation & Maintenance
###QA / Testing / Compatibility
Developer is responsible for ensuring that the final product meets the specifications laid out in this document and functions without errors. However, Developer is only responsible for QA testing on a 2014 Macbook Pro running OS X 10.10.3 using Google Chrome web browser. Testing on other platforms and browsers is the responsibility of the Client, and Developer makes no guarantee that the product will function on other platforms, browsers, or configurations.

###Delivery and Hosting of Files
Files will be available in a .zip file downloadable at https://github.com/mliq/mliq.github.io/tree/week4. Client is responsible for hosting and configuration on server. Developer is not responsible for load-testing or any issues caused by the server, hosting, DNS or connectivity. 

###Maintenance
No updates or maintenance are included in the scope of this agreement.
